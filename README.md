*複製檔案，若發生異常，先等一下，然後再來嘗試一次，直到成功複製為止。
*另外，只計算檔案傳送速率，不計算等待時間，
  *當要執行c# file.CopyTo ...開始計時
  *當c# file.CopyTo 發生異常時，暫停計時。
  *最後會得到總時間，最後就是總容量/總時間，即為傳輸速度。
